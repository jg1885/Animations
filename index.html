Open & Close Button With Hidden Text:

This is a button that, when clicked, reveals a hidden text section! The button also triggers a box-shadow css3 animation via keyframes, and the button text changes to reflect the open & closed state.

Random Red Squares

At first glance, there seems nothing particularly special about these red squares. However, this is a 2d rendering that gets fired when the page loads. After each page refresh or page load, a new order will be presented. This is done using HTML5 Canvas and Vanilla Javascript.
